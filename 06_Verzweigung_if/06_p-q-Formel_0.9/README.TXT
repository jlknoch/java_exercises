Zweck:   Lösen einer quadratischen Gleichung in Normalform
Autor:   Matthias Ebert
Datum:   20.12.2015
Version: 0.9


AUFGABEN

1. Testen Sie das Programm mit folgenden Koeffizienten.
   a) p = 3 und q = -2 
   b) p = 2 und q = 1
   c) p = 3 und q = 7
   
2. Führen Sie das Programm schrittweise im Debugger aus.
   Beobachten Sie die Anzeige der lokalen Variablen.
   
3. Erweitern Sie das Programm so, dass statt "NaN" angezeigt wird:
   "Die Gleichung hat keine reellen Lösungen."
   
4. Erweitern Sie das Programm nochmals.
   Für p = 2 und q = 1 soll angezeigt werden:
   "x1 = x2 = -1.0"
