Zweck:   Umwandeln von Notenpunkten in eine Zeugnisnote
Autor:   Matthias Ebert
Datum:   20.12.2015
Version: 0.1


AUFGABE

Schreiben Sie ein Programm, das es dem Benutzer ermöglicht, eine
dezimale Note mit zwei Nachkommastellen) einzugeben, und den
zugehörigen Zeugnistext ("sehr gut", "gut", ...) ausgibt.

Zusatz: Erweitern Sie das Programm so, dass es dem Benutzer ermöglicht,
mehrere Einzelnoten einzugeben, daraus den (ggf. gewichteten) Mittelwert
berechnet und die Zeugnisnote wie oben beschrieben als Text ausgibt.
